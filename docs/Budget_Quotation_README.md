# Budget Quotation — Generator

This script generates a PowerPoint presentation `Budget_Quotation.pptx` with a sample budget in Indian Rupees (₹).

Prerequisites
- Python 3.8+
- Install `python-pptx`:

```powershell
pip install python-pptx
```

Run

```powershell
python scripts\generate_budget_presentation.py
```

The script saves `Budget_Quotation.pptx` in the current working directory. Edit `scripts/generate_budget_presentation.py` to adjust line-item costs or project name.

Notes
- The numbers are example estimates. Confirm and update line items to match your project's exact requirements.
